Exercício: Números Primos

Escreva uma programa que recebe como entrada um número inteiro (n) positivo e imprime True (booleano), caso n seja um número primo, ou False (booleano), 
caso n não seja um número primo.

INFO.: Números primos são os números que têm apenas dois divisores diferentes: ele mesmo e o número um. Portanto, o número um (1) não é considerado um 
número primo, pois só possui um divisor, ele mesmo.

Exemplos:
  >>> para n = 3
  resultado: True
  
  >>> para n = 5
  resultado: True
  
  >>> para n = 1
  resultado: False
  
  >>> para n = 4
  resultado: False

---
ATENÇÃO: UTILIZE APENAS 2 ESPAÇOS PARA INDENTAÇÃO DO SEU CÓDIGO.
